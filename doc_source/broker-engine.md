# Engine<a name="broker-engine"></a>

A *broker engine* is a type of message broker that runs on Amazon MQ\. 

Amazon MQ supports the following versions of ActiveMQ:
+ ActiveMQ 5\.15\.12 \(recommended\)
+ ActiveMQ 5\.15\.10
+ ActiveMQ 5\.15\.9
+ ActiveMQ 5\.15\.8
+ ActiveMQ 5\.15\.6
+ ActiveMQ 5\.15\.0
